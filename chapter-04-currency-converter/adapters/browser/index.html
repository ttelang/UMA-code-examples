💱 Currency Converter

Convert currencies using WebAssembly (WASI)

How it works: This demo runs your Rust currency converter compiled to WebAssembly (WASI) directly in the browser. The JavaScript adapter provides a minimal WASI polyfill for stdin/stdout I/O.

To run: Build the WASM module with cargo build --bin fx --target wasm32-wasip1 --release and copy target/wasm32-wasip1/release/fx.wasm to this directory.
